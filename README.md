# E-Commerce Project

This is a **full-stack e-commerce web application** developed using **Laravel**, designed to manage products, categories, orders, and customers efficiently. The system provides a clean and responsive interface for both users and administrators.

## Features

### User Features
- Browse products by category
- View product details
- Add products to cart
- Place orders
- User authentication (register/login)
- Search products

### Admin Features
- Dashboard overview
- Manage products (create, update, delete)
- Manage categories
- View and manage orders
- Manage users
- Role-based access control

## Technologies Used
- **Backend:** Laravel 12 (PHP Framework)
- **Frontend:** HTML, CSS, JavaScript, Bootstrap
- **Database:** MySQL
- **Version Control:** Git & GitHub
- **Other:** Font Awesome, Poppins Font

## Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/athumaniMfaume/ecommerce-project.git

2. Navigate to the project directory:

   cd ecommerce-project


3. Install dependencies:

   composer install
   npm install
   npm run dev


4. Set up your .env file (copy from .env.example) and configure database settings.

   Run migrations and seeders:

   php artisan migrate --seed


5. Start the development server:

   php artisan serve


6. Open the application in your browser at http://127.0.0.1:8000.

7. Usage

   Users can browse products, add them to cart, and checkout.

   Admin can log in to manage products, orders, and users from the dashboard.
