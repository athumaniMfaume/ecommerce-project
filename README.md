# E-Commerce Project

This is a **full-stack e-commerce web application** developed using **Laravel 12**, designed to manage products, categories, orders, and customers efficiently. The system has **two user roles**: Admin and Normal User, each with specific access and duties.  

- **Admin**: Manages the system, products, categories, users, and orders.  
- **Normal User**: Can browse products, add to cart, and place orders.

## Features

### User Features
- Browse products by category  
- View product details  
- Add products to cart  
- Place orders  
- User authentication (register/login)  

### Admin Features
- Dashboard overview  
- Manage products (create, update, delete)  
- Manage categories  
- View and manage orders  
- Manage users  
- Role-based access control

## Installation

1. Clone the repository:
```bash
git clone https://github.com/athumaniMfaume/ecommerce-project.git

2. Navigate to the project directory:
     cd ecommerce-project


3. Install PHP dependencies:
     composer install


4. Install Node.js dependencies:
     npm install
     npm run dev


5. Configure your .env file (copy from .env.example) and set your database credentials.
      Run migrations and seeders:
      php artisan migrate --seed


6. Start the development server:
     php artisan serve


6. Open the application in your browser at http://127.0.0.1:8000.

7. Usage
    Admin can log in using the admin credentials to manage the system.
    Normal users can register, browse products, add them to the cart, and place orders.
